# Sample Java Project for Students

This is a beginner-friendly Java project designed to teach fundamental programming concepts.

## 📁 Project Structure

```
sample-java-project/
├── HelloWorld.java      # Basic OOP concepts
├── Calculator.java      # Methods and error handling
└── README.md           # This file
```

## 🚀 Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher installed
- Text editor or IDE (VS Code, IntelliJ, Eclipse, etc.)

### Compilation

Compile the Java files:

```bash
# Compile HelloWorld.java
javac HelloWorld.java

# Compile Calculator.java
javac Calculator.java
```

### Running the Programs

Run the compiled programs:

```bash
# Run HelloWorld demo
java HelloWorld

# Run Calculator demo
java Calculator
```

## 📚 Learning Concepts

### HelloWorld.java Covers:
- ✅ Class definition and structure
- ✅ Object instantiation (creating objects)
- ✅ Instance variables and static variables
- ✅ Constructors (initialization methods)
- ✅ Method definitions
- ✅ The `main` method (program entry point)
- ✅ String concatenation and output

### Calculator.java Covers:
- ✅ Static methods (class methods)
- ✅ Method parameters and return types
- ✅ Mathematical operations
- ✅ Exception handling (try-catch blocks)
- ✅ Javadoc comments and documentation
- ✅ Conditional logic (if statements)
- ✅ Using Java's built-in Math class

## 🎯 Next Steps for Students

1. **Modify HelloWorld.java**
   - Add more student objects
   - Create a method to calculate average age
   - Add additional properties (e.g., GPA, major)

2. **Extend Calculator.java**
   - Add more mathematical operations (square root, absolute value, etc.)
   - Create a calculator menu for user input
   - Implement percentage calculations

3. **Create New Classes**
   - Create a `BankAccount` class with deposit/withdraw methods
   - Create a `Student` class with grades and GPA calculation
   - Create a `Rectangle` class with area and perimeter methods

4. **Version Control Practice**
   - Commit changes to Git
   - Create feature branches
   - Practice collaborative development

## 💡 Tips for Teaching

- Start with HelloWorld.java to introduce OOP concepts
- Use Calculator.java to show method design and error handling
- Have students run and modify the code
- Encourage students to add comments explaining what they learn
- Use these as templates for student assignments
